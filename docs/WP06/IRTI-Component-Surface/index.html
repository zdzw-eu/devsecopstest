    General Description

    This app is based on IR Thermal inspection. The app will allow end-users to monitor a material or a component condition. This monitoring is based on the following:

    1. Surface visualization
    2. Automatic defect detection based on AI.

    Architecture Diagram

    The solution includes the following components: - IR Thermal Inspector. This element is the core of the app. It receives the stream of the camera, the AI-model and their configurations to perform the inference to automatically detect the surface defects. It communicates with GUI (Display) through OPC-UA and with the camera using the GeniCam standard. - GUI (Display): with the real-time image. - Cameras. They will acquire the images and send them to the Inspector. It communicates with the former element using GeniCam standard. - Storage. It will store the information of the defects detected. - AI Training Module. It represents the place in which the models are developed and the sent to the Inspector.



    Image Overview

    The App comprises a graphical user interface (GUI) composed of five windows or panels that allow interaction with the files and functionalities of the application. These windows can be selected in the left banner marked in the following figure and are divided into ‘Menu’, ‘IR Thermal Inspector’, ‘Analysis’, ‘New User’ and ‘Settings’. The ‘Menu’ window, allows the user to select the different sections in the app.



    The application interface comprises a comprehensive control center, enabling the management of the automatic inspection system. Users can access an offline view of the IR cameras on “IR Thermal Inspector”, as well as the most recent image that has undergone inference processing. Additionally, the interface features convenient buttons to perform the main tasks.

    There are displayed the principal tools to visualize images, download, zoom. It is possible to reduce noise with the preprocessing button. It allows to infer with a tiny or large model, and launch the thermal inspector to infer on new images. It could be interesting to display the type of defects by class.



    In the figure bellow, a separated window is displayed to show the analysis screens i.e. their location in 2D images or graph of the history of the last 10 images analyzed. The main idea in this screen will be to know the correlations between the sensor data and the defects in the images. An image of the torch position will be provided, in overall percentage may be an option. It will also be possible to download a report with the exported results.



    Finally, the ‘New User’ window allows adding new users to manage the application. The ‘Settings’ page includes the main configuration features to work with the software, such as security, privacy and display settings and preferences.

    How To Use

    The instructions to follow for the use of the app will be defined once the application has been developed. The following can be established as a provisional guideline: 1. Installation of the app (to be developed). 2. Select the 'New user' section in the 'Menu' window of the app and register. 3. Select the 'preprocessing and visualization' section in the 'Menu' window of the app. 4. Push the 'Load image' button to select the image to be analyzed. 5. In the 'preprocessing and visualization' window, scroll through the slices and edit the settings (birghtness, contrast, transparency, speckle reduction) of the image. 6. Use the 'Anaylsis' window to assess the defect detection and classification of the IR Thermal image. Optionally, export the results of the analysys via the button 'Export results'.
