    MIR-OCT-2D

    General Description

    This app is based on MIR-OCT inspection. The app will allow end-users to monitor a material or a component condition. This monitoring is based on the following: i. Surface and sub-surface visualization ii. Automatic defect detection based on AI.

    Top Ten Functionalities

    1. *OCT Visualization*
    2. *CNN defect localization*
    3. *Contrast-Brightness*
    4. *Speckle Reduction/Smoothing*
    5. *Transparency*
    6. *XY Slice Inspection*
    7. *YZ Slice Inspection*
    8. *XZ Slice Inspection*
    9. *Scale bars*
    10. Volume Scrolling

    Architecture Diagram

    The architecture diagram is shown in the figure. Through the GUI and its various screens, the actor communicates with the 'Defect Inspector' module, uploading the images in PNG format obtained by the MIR-OCT system. This module communicates with the preprocessing module to adjust the parameters of the images, which are sent to the AI module for defect detection and classification. The results are produced and analyzed in the Defect Inspector module, and the AI model for defect detection and classification and the results are kept in the storage.



    Image Overview

    The App comprises a graphical user interface (GUI) composed of five windows or panels that allow interaction with the files and functionalities of the application. These windows can be selected in the left banner marked in the figure bellow and are divided into ‘Menu’, ‘Preprocessing and visualization’, ‘Analysis’, ‘New User’ and ‘Settings’. The ‘Menu’ window will allow the user to select the different sections in the app.



    The ‘Preprocessing and visualization’ window of the app at its current stage is shown in the figure. At the top of the panel is a ‘Load OCT’ button to upload the selected volume the user wants to show. This window consist of visualization tools that comprises: - XY, YZ and XZ slices --to search specific dimensions for interfaces and objects in the volume. - Slider bars -- To scroll the slices and visualize them along the volume. - Brightness, Contrast, Speckle reduction and Transparency – to manipulate voxels values and suppress speckle noise for increasing local structure contrast in volume.

    The ‘Analysis’ panel will show the tools to assess the defect detection and classification of the OCT volumes. In addition to visualizing the OCT volume in 3D, this screen shows the type of defects by class, their location in 2D images and a graph of the history of the last 10 OCT volumes analyzed.

    At the top of the panel is a ‘Load file’ button to upload the file the user wants to analyze. The red rectangles in the figure mark the AI defect detection and classification displays (ii). Different categories of defects can be identified automatically in the volume. Examples of defect types are: - Voids - Pores - Others (cracks and irregular fractures)

    At the bottom of the panel is the option of exporting the results by pushing the ‘Export results’ button.

    Finally, the ‘New User’ window allows adding new users to manage the application. The ‘Settings’ page includes the main configuration features to work with the software, such as security, privacy and display settings and preferences.

    Hardware Components

    Edge processing unit based on GPUs: AI inference optimized on available GPU hardware with 100% high-rate inspection.

    How To Use

    The instructions to follow for the use of the app will be defined once the application has been developed. The following can be established as a provisional guideline: 1. Installation of the app (to be developed). 2. Select the 'New user' section in the 'Menu' window of the app and register. 3. Select the 'preprocessing and visualization' section in the 'Menu' window of the app. 4. Push the 'Load OCT' button to select the volume to be analyzed. 5. In the 'preprocessing and visualization' window, scroll through the slices and edit the settings (brightness, contrast, transparency, speckle reduction) of the volume/slices. 6. Use the 'Anaylsis' window to assess the defect detection and classification of the OCT. Optionally, export the results of the analysys via the button 'Export results'.
